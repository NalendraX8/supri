## 🛠️ Getting Started

### Prerequisites
* Flutter SDK (Targeting iOS 15+ & Android 9+)
* Dart SDK

### Installation
1. Fetch packages:
   ```bash
   flutter pub get
   ```
2. Run unit and widget tests:
   ```bash
   flutter test
   ```
3. Launch development build:
   ```bash
   flutter run
   ```
