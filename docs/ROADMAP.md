# ROADMAP.md - Supri POS Project Roadmap

## Phase 1: MVP Core POS (Current Focus)
* Core cashier login and multi-outlet select.
* Local database storage and product catalog sync.
* Cart checkout support for Cash & QRIS payment.
* Cash shift tracking (Opening, movement, closing).
* Automated transaction queue syncing.

## Phase 2: Receipts & Peripheral Support
* ESC/POS Bluetooth thermal receipt printing (58mm/80mm size configurations).
* Cash Drawer opening signal command integration.
* Barcode scanner scanning integration using mobile camera.

## Phase 3: Reporting & Analytics
* Deep shift reports detailing cashier performance metrics.
* Cloud-based owner panel supporting remote catalog edits and visual sales charts.
* Transaction export to CSV/PDF.
