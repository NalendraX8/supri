# TECH_STACK.md - Supri POS Tech Stack Details

## 1. Core Framework
* **Flutter SDK:** Cross-platform compiler targeting iOS and Android platforms.
* **Dart Programming Language:** Clean static typing supporting asynchronous futures and streams.

## 2. State Management & DI
* **flutter_bloc:** For scalable UI-logic separation.
* **get_it:** Service Locator providing low-overhead dependency injection container.
* **equatable:** Simplifies value equality logic on states and models.

## 3. Storage & Network
* **Isar Database:** High performance offline NoSQL database for local caching.
* **http:** Simple HTTP connection handler.
* **connectivity_plus:** For network status tracking.

## 4. UI Utilities
* **flutter_screenutil:** Standard scaling helper.
* **gap:** Clean spacing widget replacing traditional sized-box blocks.
* **google_fonts:** Dynamic fonts loading (Inter, Roboto).
